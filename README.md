# ToDo List 应用

一个现代化的待办事项管理应用，基于 Next.js + TypeScript + Tailwind CSS 构建。

## ✨ 功能特性

- ✅ 添加、编辑、删除待办事项
- ✅ 标记任务完成状态
- ✅ 筛选任务（全部/进行中/已完成）
- ✅ 任务统计显示
- ✅ 本地数据持久化（localStorage）
- ✅ 响应式设计，支持移动端

## 🎨 设计特点

- 现代化 UI 设计
- 橙红色暖色调渐变（避开蓝紫色系）
- 流畅的动画过渡效果
- 清晰的视觉层次

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 运行开发服务器
npm run dev

# 打开浏览器访问
# http://localhost:3000
```

## 🛠️ 技术栈

- **框架**: Next.js 16 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **状态管理**: React Hooks (useState, useEffect)
- **数据存储**: localStorage

## 📝 项目结构

```
todo-app/
├── app/              # Next.js App Router
├── components/       # React 组件
├── types/           # TypeScript 类型定义
└── public/          # 静态资源
```
